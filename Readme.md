#Como utilizar
##Cmake Tools no VS Code:
1. Ctrl+Shift+p: Quick Start
2. Ctrl+Shift+p: Selecionar Kit GCC ou Clang
3. Ctrl+Shift+p: Configure
4. Compilar

##Via terminal
1. mkdir build
2. cd build
3. cmake ..
4. make -j
5. sudo make install
